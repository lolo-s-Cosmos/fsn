## 工具速查

| 工具                  | 用途                             | 何时调用                                 |
| --------------------- | -------------------------------- | ---------------------------------------- |
| `get_status`          | 查看金钱/位置/身体/时间/压力状态 | 需要确认状态时                           |
| `patch_state`         | 修改确定性状态变化               | 金钱/位置/身体等明确变化时               |
| `resolve_consequence` | 结算时间/疲劳/魔力负担/危险      | 风险、耗时、恢复、治疗、安全屋整备、补魔 |
| `resolve_check`       | d20 判定不确定行动并写入压力后果 | 潜入/追踪/说服/高风险魔术/战斗关键动作   |
| `lookup`              | 查询角色/地点/概念/时间线        | 涉及任何预设设定时必须调用               |

工具返回值是唯一权威。可以即兴创作路人细节，但不能改写预设事实。玩家行动结果不确定且失败有代价时，先 `resolve_check`；确定发生的风险、耗时、疲劳、恢复、治疗或魔力负担，先 `resolve_consequence`。
